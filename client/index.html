API: Buscador de Imágenes

Historias de usuario:
  • Puedo obtener la URL de la imagen, texto alternativo (alt) y url de la página web relativo a una cadena de caracteres dada.
  • Puedo elegir la página de respuesta agregando ?offset=[numero-pagina] como parametro a la URL.
  • Puedo obtener una lista de las busquedas realizadas más recientes.

Ejemplo de búsquedas:

https://floating-headland-39020.herokuapp.com/api/imagesearch/gato%20perro
https://floating-headland-39020.herokuapp.com/api/imagesearch/gato%20perro?offset=10
